🎮 Simon Says Game

A fun and interactive memory game built with HTML, CSS, and JavaScript.
The game challenges players to follow and repeat an increasing sequence of colors and sounds.

🚀 Features

🔊 Sound and color feedback for each button

🎯 Increasing difficulty as the sequence grows

⏱️ Tracks your progress and resets on a wrong move

🎨 Simple and colorful UI

🛠️ Tech Stack

HTML – Structure of the game

CSS – Styling and animations

JavaScript – Game logic and interactivity


🎮 How to Play

Press any key to start the game.

Watch the sequence of flashing colors.

Repeat the sequence by clicking the buttons in the same order.

Each level adds one more step to the sequence.

If you click the wrong button → Game Over!
